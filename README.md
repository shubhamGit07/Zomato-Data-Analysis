# 🍽️ Zomato Data Analysis Dashboard

## 📊 Project Overview
This project analyzes Zomato restaurant data using Python and Streamlit.

## 🚀 Features
- Data Cleaning using Pandas
- Interactive Dashboard using Streamlit
- Filters for Restaurant Type and Online Orders
- KPI Metrics (Ratings, Votes)
- Data Visualization using Matplotlib & Seaborn

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib
- Seaborn
- Streamlit

## 💡 Insights
- Online ordering improves ratings
- Most ratings are between 3.5 - 4.5
- Cost does not always affect ratings
